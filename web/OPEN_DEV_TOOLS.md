# How to Open Developer Tools (When F12 Doesn't Work)

## Method 1: Right-Click Menu
1. **Right-click anywhere on the page**
2. Select **"Inspect"** or **"Inspect Element"**
3. Developer Tools will open

## Method 2: Browser Menu
### Chrome/Edge:
1. Click the **3 dots menu** (top right)
2. Go to **More Tools**
3. Click **Developer Tools**

### Firefox:
1. Click the **3 lines menu** (top right)
2. Go to **More Tools**
3. Click **Web Developer Tools**

## Method 3: Keyboard Shortcuts
- **Ctrl + Shift + I** (Windows/Linux) - Opens DevTools
- **Ctrl + Shift + J** (Chrome/Edge) - Opens Console directly
- **Ctrl + Shift + K** (Firefox) - Opens Console directly

## Method 4: Address Bar
1. Type in address bar: `chrome://inspect` (Chrome) or `edge://inspect` (Edge)
2. Or type: `about:debugging` (Firefox)

## What to Look For:
Once DevTools is open:
1. Go to **Console** tab
2. Look for **RED error messages**
3. Copy any error messages you see
4. Share them to get help fixing the issue
