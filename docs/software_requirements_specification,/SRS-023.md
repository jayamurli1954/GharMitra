# SOFTWARE REQUIREMENT SPECIFICATION  
System: GharMitra  
Module: Mobile App Wrapper (PWA-Lite)  
Derived From: 6-01-07-008-LITE  

---

## 1. PURPOSE  
Provide a mobile-app-like experience for GharMitra using PWA technology without changing backend logic, accounting, or data flow.

---

## 2. SCOPE  
This SRS covers:
- Installable mobile app
- Full-screen display
- Mobile UI layout

This SRS does NOT include:
- Offline data
- Push notifications
- Local storage
- Sync

---

## 3. FUNCTIONAL REQUIREMENTS  

| ID | Requirement |
|----|-------------|
| FR-1 | App must be installable on Android and iOS |
| FR-2 | App must open full-screen |
| FR-3 | App must display icon & splash screen |
| FR-4 | All pages must be mobile-friendly |

---

## 4. NON-FUNCTIONAL  

- Must work on Chrome, Safari, Edge  
- Must use HTTPS  
- Must not store business data locally  

---

## 5. TECHNICAL COMPONENTS  

- manifest.json  
- service-worker.js (static cache only)  
- Mobile CSS  

---

## 6. SUCCESS  

- Users install app  
- No browser bar  
- Same backend behavior
