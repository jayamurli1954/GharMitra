## 1. CHANGE SUMMARY

### Current Situation

GharMitra is a browser-based web app. On mobile:

* It opens inside Chrome/Safari
* Browser bars are visible
* No home-screen icon
* It does not feel like a real app

### Requested Change

Convert GharMitra into a **Simple Installable Mobile App (PWA-Lite)** so that:

* Users can install it on Android & iPhone
* It opens full-screen without browser bars
* It has an app icon and splash screen
* It uses the same backend and web pages
* No offline or sync logic is added

### What is explicitly NOT included

* No offline mode
* No push notifications
* No local data caching
* No background sync

This is a **presentation layer upgrade only**.
